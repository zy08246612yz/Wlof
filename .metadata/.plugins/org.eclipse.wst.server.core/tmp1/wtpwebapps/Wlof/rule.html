游戏规则

1.游戏背景

在一个美丽的小村庄里,潜伏着可怕的狼人!每天晚上.都有一个善良的村民称为狼人的牺牲品,但是,村民们不会坐以待毙,视图在白天找到这些狼人并处决他们.

2.游戏人数

8-18人.

3.游戏角色:

  • 普通村民
  • 狼人
  • 预言家
  • 女巫
  • 猎人
  • 小女孩
  • 守卫
  • 丘比特

4.阵营

  第三方阵营

当丘比特指定的情侣为好人和狼人时,丘比特及两名情侣组成第三方阵营.第三方阵营藏匿于好人与狼人中间.

  狼人阵营

所有除第三方阵营外,拿到狼人牌的玩家.

  好人阵营

所有除第三方阵营外,拿到非狼人牌的玩家.

5.游戏目标

  第三方阵营

杀死或者处决掉所有狼人和好人.

  狼人阵营

杀死或者处决掉所有好人和第三方阵营.

  好人阵营

处决掉所有狼人.
